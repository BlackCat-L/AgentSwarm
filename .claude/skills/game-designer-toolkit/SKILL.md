# 游戏策划工具箱

将游戏创意转化为专业策划文档和系统的完整工具箱。适用于独立开发者、游戏策划和工作室。

## 快速开始

**新项目？** 从这里开始：
1. 用几句话告诉我你的游戏概念
2. 我会生成完整的 GDD 大纲
3. 我们一起完善每个章节

**触发示例：**
- "帮我设计一个 roguelike 卡牌构筑游戏"
- "为我的手机解谜游戏创建一份 GDD"
- "为我的 RPG 设计经济系统"

## 核心能力

### 1. GDD 生成器

创建包含业界标准章节的专业 GDD：

- 游戏概述与独特卖点
- 目标用户与市场分析
- 核心玩法循环
- 机制与系统
- 角色与故事
- 关卡设计框架
- 美术与音频方向
- 技术需求
- 开发路线图

完整模板结构见 `references/gdd-template.md`

### 2. 游戏系统设计

设计平衡、有吸引力的游戏系统：

| 系统类型 | 关键要素 |
|---------|---------|
| **经济** | 货币类型、产出/消耗、通胀控制 |
| **战斗** | 伤害类型、平衡曲线、技能树 |
| **成长** | 经验曲线、解锁节奏、精通深度 |
| **社交** | 多人系统、公会、交易 |
| **商业化** | 免费友好、付费、混合模式 |

详细模式见 `references/system-design.md`

### 3. 关卡与世界设计

创造引人入胜的空间和节奏：

- 关卡布局原则、流程引导、视觉提示
- 难度曲线：学习→掌握→挑战
- 世界构建：区域、兴趣点、探索路径
- 节奏控制：紧张/放松循环、叙事节奏

框架见 `references/level-design.md`

### 4. 叙事与任务设计

打造引人入胜的故事和目标：

- 故事结构（三幕式、英雄之旅、分支叙事）
- 任务设计模式（收集、护送、防守、探索）
- 对话系统与分支
- 世界观与一致性

### 5. 游戏提案

面向不同受众的快速提案：

- 30秒电梯演讲
- 核心循环 + 差异化 + 行动号召

## 工作流程

### 新游戏概念
1. 概念捕捉 → 收集核心想法、类型、目标平台
2. GDD 生成 → 创建完整文档结构
3. 系统设计 → 定义核心机制和循环
4. 迭代完善 → 逐章打磨
5. 提案创建 → 提炼为演示文稿

### 已有项目
1. 审计 → 审查现有策划文档
2. 缺口分析 → 识别缺失元素
3. 平衡审查 → 检查系统交互
4. 文档化 → 更新并标准化

## 按类型指导

### RPG
- 角色成长深度、分支叙事、装备掉落系统、伙伴机制

### FPS/动作
- 武器多样性与平衡、敌人AI模式、地图流程与关键点

### 策略
- 资源管理、科技树与解锁、非对称阵营、胜利条件

### 手游/休闲
- 单局时长优化、成长钩子、变现整合、无障碍设计

### Roguelike/Roguelite
- 单局变化与重玩性、 meta 成长、构筑多样性、难度缩放

## 使用技巧

1. 明确游戏类型 — "动作RPG" 比只说 "游戏" 好得多
2. 定义目标用户 — 休闲？硬核？年龄段？
3. 说明目标平台 — 手机/PC/主机影响设计决策
4. 提供参考作品 — "像 X 遇到 Y" 帮助对齐方向
5. 逐章迭代 — 比一次性全部输出效果好
